# My most common response to help requests

Having worked and studied alongside many others learning to program, the answer I give most frequently is "Well let's look at the data flow".

Personally for myself I emphasis this because it helps me understand my objective, and understanding the flow is a transferable skill whether you're learning Angular, React, Backb- ... Vue, or writing server side authentication or database code or almost anything else. -->
